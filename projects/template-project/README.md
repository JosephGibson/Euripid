# template-project/

Canonical bootstrap project for Euripid. New users should copy this directory to `projects/<new-project-name>/`, rename the project metadata inside `project.config.json`, then edit the copied project's environments, profiles, pages, flows, scenarios, and data in place. This directory is also the committed harness-validation target for the current repo layout.

The template ships with five starter profiles under `profiles/`: `smoke`, `debug`, `load`, `soak`, and `breakpoint`.
